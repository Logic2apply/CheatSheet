  2. Font Style

    • We can use the technique of font stack. A font stack is a list of fonts that are listed in order of preference you would like them to appear in case some fonts are not loading. The example of this is shown below- 

      p {font-family:'Ubuntu', 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;}

      This list will be iterated until the specified font is not available in the system.

      To import the code, there is no website better than Google Fonts To use it, simply copy the style-sheet and add it to your code and update the font stack with the specific font you desire.

    • Font Size is used to set the size of a font. In the above example, we used our font size to be 33px. Pixel 'px' is the unit of the font size and it is 1/96th of an inch. 

      p{
    font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
    font-size: 36px;
}

    • Line-height is the spacing between the fonts (current font and previous font). 

      p{
    font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
    font-size: 36px;
    line-height: 1.5rem;
}

    • The font weight property sets how thick or thin character in text should be displayed. 

      p{
    font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif;
    font-size: 36px;
    line-height: 1.5rem;
    font-weight: 600; /* or bold, bolder, light, lighter and 100-900*/
}

  5. CSS Box Model, Margin and Padding

    The box model is a very important topic of CSS and if not understood properly, it can create a lot of confusion in the future. It is the basic framework of web development whether you are making a website using any other language such as Angular or React. The box model helps us to define the padding,border, and margin around an element. So from the above diagram we can see where all these things lie around the element. The element is in the center surrounded by padding, border and margin.

    These parts can be explained as-

    • Content-content of the box, where text and images appear.
    • Padding-It clears an area around the content. The padding is transparent.
    • Border-A border is one that covers the padding and content.
    • Margin-It clears an area outside the border. The margin is also transparent.

    Image Preview

    • There is padding or margin shorthand for all directions. The first value is for top, 2nd value is for the bottom, 3rd value is for left and 4th value is for right. 

      padding: 23px 56px 6px 78px;
/* padding: top right bottom left; */
margin: 23px 56px 6px 78px;
/* margin: top right bottom left; */

    • There is another technique for using the shorthand technique if you want to give the same values for left/right and top/bottom. The first value is the same for both the top and bottom and the second value is the same for both left and right. The two values can be represented as x and y values. 

      padding: 23px 56px;
/* padding: y(top/bottom) x(left/right); */
margin: 23px 56px;
/* margin: y(top/bottom) x(left/right); */

      Border radius is used to apply an arc type shape in each corner of the border.

    • If we want to set all the values of padding and margin differently, we can write code like this. 

      padding-top: 79px;
      padding-bottom: 79px;
      padding-left: 34px;
      padding-right: 79px;

      margin-top: 3px;
      margin-bottom: 5px;
      margin-left: 34px;
      margin-right: 5px;

    • On giving width to the element and after that applying padding in the container, the width also changes. It is because in the actual width of an element, margin is already been added into it. If you want this not to happen then you can use the property of ‘box-sizing’. 

      box-sizing: border-box;

      Now if you change the padding then it will not adjust the width according to the padding.

      We can take the help of universal selector in the CSS to apply the property of box-sizing in all the elements available. It is denoted with a ‘*’. 

      *{
    box-sizing: border-box;
    margin: 0px;
    padding: 0px;
}
  6. Float and Clear

    The CSS float property specifies how an element should float. The CSS clear property specifies what elements can float beside the cleared element and on which side. The float property is used for positioning and formatting content, for example, let an image float left to the text in a container. The float property can have one of one of the following values-

    • Left- The elements floats to the left of its container.
    • Right- The elements floats to the right of its container.
    • None- The element does not float (it will be displayed just where it occurs in the text). This is default.
    • Inherit- The element inherits the float value of its parent.
    float: left;

    To make object float in left we will add this property as shown above. 

    float: right;

    To make object float in right we will add this property as shown above.

    But now the condition arises that the float object overlaps the other lying below it. To overcome this situation we use clear property and it will solve our problem.

    To make that the left floating object doesn't overlap its underlying object we will use-

    clear: left;

    To make that the right floating object doesn't overlap its underlying object we will use-

    clear: right;

    To make that the both the right and left floating object doesn't overlap its underlying object we will use-

    clear: both;
    For Text Alignment we have-
    text-align: right;
    text-align: left;
    text-align: center;
    text-align: justify;

    What's the need to style links and buttons?

    As the HTML gives us very ugly links and buttons we style it using CSS by making some utility classes for further use and it is also recommended if we are going to make production level website.

    For instance our link and buptton will be like-

    <a href="https://yahoo.com" class="btn">Read more</a>
<button class="btn">Contact us</button>

    Till here the buttons will look very ugly to make them attractive we will add styling to it like- 

    .refer-btn{
    color: white;
    background-color: black;
    border-radius: 26px;
    margin: 4px auto;
    padding: 6px 10px;
    transition-property: all;
    transition-duration: 0.3s;
    transition-timing-function: ease-in-out;
    text-decoration: none;
    text-align: center;
    font-family: 'Roboto', sans-serif;
    font-weight: 300;
    display: block;
    width: 12rem;

}

    and if hovered

    .refer-btn:hover{
    color: rgb(0, 0, 0);
    background-color: rgb(255, 255, 255);
    border: 2px solid black;
}
    Let us now see what Pseudo Selectors are. A pseudo class is used to define a special state of an element.

    1. Hover is used to change the color of text or background of a button as soon as you hover that part. The code for this is as below.

      .refer-btn:hover{
    color: rgb(0, 0, 0);
    background-color: rgb(255, 255, 255);
    border: 2px solid black;
}

    2. The next Pseudo selector is Visited. As soon as you visit the anchor tag button and click the link mentioned, it changes its color. To apply this property, write the code as follows-

      a:visited{
    background-color: cornsilk;
    color: rgb(6, 56, 73);
}

    3. The next selector is Active. If you visit any button, and click it, it becomes active and showcases with different properties. The code for this is-

      a:active{
    background-color: rgb(0, 0, 0);
    color: rgb(193, 239, 255);
    border-radius: 25px;
}

  8. Display Property

    By inspecting both the elements in the Chrome browser, we see that the image is an inline element and the h3 heading is the block element. Our objective is to bring all the elements to the center of the webpage. We can achieve it by adjusting the width of the block element i.e. the heading. The respective code of the following is-

    header{
    border: 2px solid red;
    margin: auto;
    width: 1200px;
}

    The display of “img” is inline and therefore, to make it come to center, we have to set the property display as block as follows-

    img{
    margin: auto;
    display: block;
    width: 34px;
}

    The next problem which arises is that when we stretch the full width of the page, the text in the heading moves towards left. So to move it towards the center, we can set the property of text-alignment as center.

    Display inline means it will take the space according to the size of the element. Display block means we can set its width and by margin manually.

    Now suppose we want to make an element inline as well as customize its width too, then in that case we can use inline-block.

    The inline-block property here allows us to change the width of inline elements also. To ensure that all the three blocks come in a single line, we can use the property box-sizing. It ensure that the width we provide is not changed including padding and margin. 

    display: block;
    display: inline;
    display: inline-block;
    NOTE: margin: auto; will not work without the element having display block.
